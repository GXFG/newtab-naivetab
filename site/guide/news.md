# 📰 资讯

> 聚合热门资讯，多平台热榜一览

资讯组件以 Tab 页签形式展示多个平台的热门榜单，让你在新标签页就能一览全网热点。

## 🖼️ 效果示意

```                                                                      
+------------------------------------------------------+                 
|  [知乎] [微博] [百度] [B站] [今日头条] ...             |                 
+------------------------------------------------------+                 
|                                                      |                 
|  1  这条新闻标题很长很长很长很长很长... HOT 123万     |                
|  2  另一条热点新闻                     HOT 98万       |                
|  3  第三条热门内容                     HOT 87万       |                
|  4  第四条内容标题                     HOT 76万       |                
|  5  第五条内容标题                     HOT 65万       |                
|  ...                                                 |                 
|                                                      |                 
|                                      [刷新]          |                 
+------------------------------------------------------+                 
```                                                                      

- 前三名序号使用不同颜色高亮
- 鼠标悬停显示完整标题（超长文本截断时）
- 左键点击新标签页打开详情，中键点击后台打开

## 📋 支持的平台

| 平台 | 榜单类型 | 说明 |
|------|----------|------|
| 知乎 | 热榜 | 知乎社区热门话题 |
| 微博 | 热搜 | 微博实时热搜 |
| 百度 | 热搜榜 | 百度搜索热搜 |
| B站 | 热门视频 | Bilibili 热门视频 |
| 今日头条 | 热榜 | 头条热点资讯 |
| 36氪 | 热榜 | 科技创投资讯 |

> 可在设置中选择展示的平台，按需开启。

## ⚙️ 设置项

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 平台选择 | 多选开关 | 选择要展示的热榜平台 |
| 刷新间隔 | 数字输入 | 自动刷新的时间间隔（分钟） |
| 字体大小 | 滑块 | 标题文字字号 |
| 字体颜色 | 颜色选择器 | 标题颜色，浅色/深色独立配置 |
| 背景色 | 颜色选择器 | 列表项背景色 |
| 组件宽度 | 滑块 | 组件整体宽度 |
| 组件高度 | 滑块 | 组件整体高度 |
| 圆角 | 滑块 | 组件外框圆角 |

## ⚡ 功能特性

- 可选择展示的平台，最多同时展示多个来源
- Tab 页签切换查看不同平台的热门内容
- 数据加载失败时提供手动刷新入口
- 自动定时刷新（可自定义间隔），保持内容最新

## 🎨 自定义

- 可调整组件宽度、高度、圆角
- 可自定义字体、颜色、背景色
- 毛玻璃背景效果
- 支持边框、阴影
- 浅色/深色主题独立配置
