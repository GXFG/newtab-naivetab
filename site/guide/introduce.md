## 💡 它是什么？

NaiveTab 新标签页，一款简洁的浏览器新标签页扩展，可以高度自定义你的新标签页。

> 可视化键盘展示书签和浏览器指令，肌肉记忆一键直达。支持自由拖拽、深度自定义外观、专注模式、云同步。

- 🧩 包含数字时钟、模拟时钟、翻页时钟、霓虹时钟、日期、万年历、天气、搜索、书签、备忘录、资讯、倒计时、年进度等组件
- ✨ 幻彩背景：极光、流体、波浪、涟漪等 8 种纯 CSS 动态光效，零性能开销
- 🎨 智能取色：根据壁纸自动提取主题色与背景色，随壁纸同步变化
- ⚡ 内置指令键盘与书签键盘功能，支持 96/98/104 等多种配列，无修饰键模式一键直达
- 🛠 可以高度定制化标签页，如：自由拖拽缩放组件；改变各组件的颜色以及其他样式；修改背景图或幻彩动效；通过鼠标右键快速进行设置，等等。

## 🧩 内置组件

### ⏰ 时间与日期

| 组件   | 说明                           |
|--------|--------------------------------|
| 🕐 时钟   | 数字、模拟、翻牌、霓虹四种风格         |
| 📅 日期   | 可自定义格式与样式                 |
| 🗓 万年历 | 农历、节假日、法定假日               |
| ⏳ 倒计时 | 自定义时长、进度环、计时提醒           |
| 📊 年度进度 | 百分比 + 点阵图，感受时间流逝         |

### 🎨 背景美化

| 功能         | 说明                                         |
|--------------|----------------------------------------------|
| 🖼️ 壁纸      | Bing 每日一图 / 本地图片 / 自定义 URL，支持视差效果 |
| ✨ 幻彩       | 8 种纯 CSS 动态光效动画，零性能开销，支持自定义配色  |
| 🎨 自动取色   | 智能分析壁纸色调，自动设置主题色与背景色           |

### 🛠 效率工具

| 组件      | 说明                               |
|-----------|------------------------------------|
| 🔍 搜索栏     | 自定义搜索引擎，快速检索               |
| 📁 文件夹书签 | 网格浏览书签文件夹                   |
| 📝 备忘录     | 记录灵感与待办                       |
| 🌤 天气      | 温湿度、风力、生活指数、异常预警         |

## 🎯 动机

> 这一切都源于个人对60配列键盘的喜爱，可以通过自己配置好的路径，完成100%键盘的功能。
