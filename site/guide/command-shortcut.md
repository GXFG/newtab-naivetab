# ⌨️ 指令键盘

> 通过键盘快捷键在浏览器任意页面快速执行标签页操作、页面控制、滚动等指令，无需鼠标，肌肉记忆一键直达

指令键盘功能让你可以将常用的浏览器操作绑定到键盘按键上，通过修饰键组合（如 `Alt + A`）在任何网页中快速触发。

## 🔧 基本使用

1. 在设置面板中进入 **指令键盘** 设置
2. 录制修饰键（如 `Alt`、`Ctrl+Shift` 等）
3. 点击虚拟键盘上的按键，为其选择要执行的指令
4. 在浏览器任意页面按下 **修饰键 + 按键** 即可触发对应指令

## 📋 指令分类

指令按功能分为以下几类（与设置面板分组一致）：

| 关闭窗口 | 关闭当前浏览器窗口 |
| 下一个标签页 | 切换到右侧相邻标签页 |
| 上一个标签页 | 切换到左侧相邻标签页 |
| 首个标签页 | 切换到窗口第一个标签页 |
| 末尾标签页 | 切换到窗口最后一个标签页 |
| 静音/取消静音 | 切换标签页音频静音状态 |
| 刷新当前窗口所有标签页 | 刷新当前窗口内全部标签页 |
| 刷新全部窗口所有标签页 | 刷新所有窗口的全部标签页 |
| 新建标签页 | 打开空白新标签页 |

| 向右移动标签页 | 将当前标签页向右移动一个位置 |

| 指令 | 说明 |

## ⚙️ 高级设置

### 修饰键

你可以自定义触发指令的修饰键组合，支持 `Alt`、`Ctrl`、`Shift`、`Meta`（Command）的任意组合。

> ⚠️ 注意：如果指令键盘的修饰键与书签键盘的全局修饰键相同，可能会产生冲突，建议设置为不同的组合。

### 🚫 域名黑名单

支持设置域名黑名单，在特定网站下不触发指令快捷键。支持以下格式：

- `google.com` — 匹配 `google.com` 及所有子域名（如 `docs.google.com`）
- `*.google.com` — 仅匹配子域名，不匹配 `google.com` 本身

> 浏览器的特殊页面（如 `chrome://`、`edge://` 等）始终不会触发指令快捷键。

### ⌨️ 输入框中触发

默认情况下，在输入框/文本域中打字时不会触发指令快捷键。开启"输入框中触发"选项后，即使在输入框中也会触发已绑定的指令，请避免设置与常用打字快捷键冲突的组合。

### 🚀 无修饰键模式

默认情况下，需要按下修饰键（如 `Alt`）再按目标键才能触发指令。**开启无修饰键模式后，只需按单个键即可直接触发**，无需任何修饰键。

> ⚠️ 注意：开启后打字时可能误触发快捷键，建议在域名黑名单中添加常用网站，或关闭「输入框中触发」选项。

### 冲突处理

如果书签键盘和指令键盘同时开启了无修饰键模式，且同一按键绑定了不同功能，**指令键盘优先执行**。

## 🎹 可视化绑定

指令键盘采用可视化键盘界面进行绑定：

1. 点击虚拟键盘上的按键选中
2. 在底部弹出的命令选择面板中选择要绑定的指令
3. 已绑定的按键会以边框高亮显示
4. 点击已绑定的按键可以修改或删除指令
