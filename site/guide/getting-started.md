# 🚀 快速上手

## 📦 首次安装

安装 NaiveTab 扩展后，打开新标签页即可看到 NaiveTab 页面。

**安装方式：**
- [官方商店安装](/install/webstore) — Chrome / Edge / Firefox 应用商店一键安装

首次安装后，默认会进入「编辑布局」模式，引导你添加第一批组件。

## ➕ 添加组件

首次安装后，默认会打开「编辑布局」。底部工具栏中列出了所有可用组件：

1. **选中**底部工具栏中的组件图标
2. **拖动**至屏幕内的任意位置，松手即可放置
3. 重复以上步骤添加更多组件

### 可用组件一览

| 名称 | 说明 |
|------|------|
| 时钟 | 数字 / 模拟 / 翻页 / 霓虹，四种风格 |
| 日期 | 可自定义格式与样式 |
| 万年历 | 农历、节假日、法定假日 |
| 天气 | 温湿度、风力、预警 |
| 搜索栏 | 自定义搜索引擎 |
| 书签文件夹 | 网格浏览书签 |
| 键盘书签 | 键盘形态展示书签 |
| 备忘录 | 笔记 / 待办 |
| 倒计时 | 进度环倒计时 |
| 年度进度 | 百分比 + 点阵图 |
| 资讯 | 多平台热榜聚合 |
| 指令键盘 | 快捷键执行浏览器命令 |

除组件外，还支持 **幻彩动态背景**（8 种纯 CSS 光效动画）和 **壁纸智能取色** 功能。

## 🗑 移除组件

在「编辑布局」下，选中屏幕内的组件后：

- **拖动**至左上角的 `垃圾桶` 区域，松手移除
- 或选中组件后按 `Delete` / `Backspace` 键删除

## ✌️ 退出编辑

- 点击左上角 `← 退出` 按钮，或按 `Esc` 键，退出「编辑布局」
- 退出后，组件进入正常展示模式

## ⚙️ 组件设置

在正常展示模式下：

- 鼠标**悬浮在组件上**，点击 `鼠标右键` 唤出设置菜单
- 点击 **组件设置** 可修改组件的功能和样式
- 在空白处点击右键可访问 **通用设置**

## 📐 再次编辑布局

退出编辑后如需调整：

- 在空白处右键 → **编辑布局**，再次进入布局编辑模式
- 在编辑模式下，可自由拖动组件调整位置，拖拽右下角手柄缩放尺寸
- 单击组件进入选中态，显示蓝色选中框和缩放手柄
- 点击或悬停被遮挡的组件可将其置顶显示
- 详细说明见 [编辑布局](/guide/edit-layout)
