# PuzzleTab 新标签页

[英文文档](https://github.com/cmd-cv/newtab-puzzletab/blob/main/README.en-US.md)

## 💡简介

一款简洁的浏览器新标签页扩展

## 🌟特性

- 包含数字时钟、模拟时钟、日期、书签、万年历、天气等功能
- 组件可放在屏幕上的任意位置（画布模式）
- 组件可定制大部分样式（字体、颜色、大小等）
- 支持根据系统外观自动切换主题（浅色模式、深色模式）
- 支持自定义背景图（Bing，本地图片）
- 同步你的数据在多个设备上（需要登录Google account）
- 支持数据导出自行备份
- 语言支持（zh-CN、en-US）
